# Cardputer ADV Cool OS

A *cool* OS I made for the M5Stack Cardputer ADV device.

## Compilation:
Clone the repo with `--recurse-submodules` and use ESP-IDF v5.4.x to compile. `Works on ESP-IDF v5.4.1, other versions are not tested yet.`

## Features:
- SD Music Player

## Plans:
- LoRa module support
- GNSS module support
- ESP-NOW transceiver
- WiFi connectivity
